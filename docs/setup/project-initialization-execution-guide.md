# Project Initialization Remaining Tasks Execution Guide

> 本文档用于一步步完成「项目初始化阶段」的剩余工程任务。\
> 仅包含可复制执行的工程步骤，不包含个人经历、进度标记或过程性记录。

------------------------------------------------------------------------

## 目标

完成以下工程闭环：

-   基础配置与环境变量加载符合约定（可复用、可迁移）
-   运行入口可稳定执行（最小可运行链路）
-   关键文档齐全且可公开（不包含私密或过程性信息）

------------------------------------------------------------------------

## 前置条件

-   你已在仓库根目录
-   你的工作分支不是 `main`
-   本地 Python 环境可用（例如 conda/venv 任一）

------------------------------------------------------------------------

## 1. 环境与依赖自检

``` bash
python --version
python -m pip --version
```

如需确认关键依赖可导入：

``` bash
PYTHONPATH=. python -c "import app; print('app import ok')"
```

------------------------------------------------------------------------

## 2. 运行入口自检（最小可运行链路）

如果你已有脚本入口（示例）：

``` bash
PYTHONPATH=. python scripts/run_agent_once.py "echo hello" --debug
```

期望：

-   进程正常退出
-   输出包含模型返回或工具返回（取决于你的 runner 逻辑）

------------------------------------------------------------------------

## 3. 执行语义回归（契约回归脚本）

运行回归脚本：

``` bash
PYTHONPATH=. python scripts/verify_execution_semantics.py
```

期望：

-   输出包含 `[PASS]` 结尾
-   样例用例均通过
-   不出现未处理的异常栈

------------------------------------------------------------------------

## 4. 文档公开检查（无个人轨迹）

请确认以下目录约定：

-   `docs/`：可公开工程文档
-   `docs-private/`：私密记录与调试输出（不对外）

如果你希望在公开文档中引用"私密目录"，请只描述目录用途，不要放任何个人过程内容。

------------------------------------------------------------------------

## 5. 收尾：格式化与快速静态检查（可选但推荐）

``` bash
python -m py_compile app/agents/runner.py app/agents/plan_executor.py app/agents/plan_validator.py
```

------------------------------------------------------------------------

## 6. 提交建议（工程风格）

建议把这类"初始化阶段剩余任务闭环"作为一个独立提交：

-   清晰的 commit message
-   包含必要的回归脚本通过证据

示例：

``` bash
git status
git add -A
git commit -m "chore(init): finalize initialization tasks and contract verification"
```

------------------------------------------------------------------------

## 完成标准

满足以下条件即可认为初始化阶段闭环完成：

-   回归脚本通过：`scripts/verify_execution_semantics.py`
-   runner 可运行：至少能跑通一条最小链路
-   公开文档不包含任何阶段性、过程性或个人轨迹信息
