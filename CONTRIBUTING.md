# Contributing Guide

感谢你对 **Jinsie AI Agent Platform** 的关注！  
本项目欢迎社区参与，但为保持架构一致性与代码质量，贡献范围有明确边界。

---

## 🟦 贡献范围说明（非常重要）

### ✔ 欢迎的贡献类型（非核心逻辑）
以下类型的轻量贡献非常欢迎：

- 文档优化（README、教程、示例）
- 拼写/格式修复
- 注释补充
- 提供示例代码
- 添加测试样例
- Issue 反馈与讨论

这些贡献不会影响主架构，是非常有价值的协作方式。

---

## ❗ 不接受的贡献类型（核心逻辑禁止外部 PR）
为保证项目的整体设计一致性，以下模块 **仅由项目维护者负责开发**：

- `app/agents/`
- `app/rag/`
- `app/pipelines/`
- `app/core/`
- 所有多 Agent 工作流
- 关键服务逻辑 / 架构设计

原因：核心模块需要统一规划、设计一致性与长期维护策略，不适合多人并行修改。  
如有想法，可以在 Issue 中提出，我们会统一评估后纳入 Roadmap。

---
